# Community Event Dashboard

This project is a simple Community Event Dashboard developed using HTML5, CSS3, JavaScript, Bootstrap 5, Core Java, and ANSI SQL concepts.

The application was created as part of frontend and full-stack learning exercises to understand responsive web design, Bootstrap components, styling techniques, JavaScript interactions, Java programming fundamentals, and database concepts using ANSI SQL.

## Features

- Responsive Navigation Bar
- Community Event Cards
- Bootstrap Grid Layout
- Registration Form
- Login Form
- Pricing Plans
- Bootstrap Buttons and Utilities
- Accordion (FAQ Section)
- Modal Popup
- Bootstrap Icons
- Responsive Design using Bootstrap

## Technologies Used
