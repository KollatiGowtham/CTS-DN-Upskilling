Welcome to the Help Center

This page explains how to use the Local Community Event Portal.

Portal Features

  • View upcoming community events
  • Register for events online
  • Submit feedback after participation
  • Watch promotional event videos
  • Save event preferences
  • Find nearby events using geolocation

Quick User Guide

Action Description
Register Fill out the registration form and submit your details.
Feedback Enter comments and suggestions after attending an event.
Location Use the location button to find nearby events.

Frequently Asked Questions

How do I register for an event?

Navigate to the Registration section, fill in the required fields, and click Register.

How can I save my preferences?

Select an event category and the portal automatically stores your preference using local storage.

How do I clear saved preferences?

Click the "Clear Preferences" button available at the bottom of the main page.
